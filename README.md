# HR-Analytics-Dashboard
Power BI HR Analytics Dashboard providing insights into employee attrition, workforce demographics, job satisfaction, and overall HR performance.


## 🚀 Project Overview

This project presents an interactive **HR Analytics Dashboard** built using Power BI to analyze employee data and uncover valuable workforce insights. The dashboard helps organizations understand employee attrition, workforce demographics, job satisfaction, and performance trends to support data-driven HR decisions.

## 🎯 Business Objectives

- Analyze employee attrition trends
- Understand workforce demographics
- Identify factors affecting employee retention
- Evaluate job satisfaction and work-life balance
- Monitor department-wise employee distribution
- Support strategic HR decision-making
